# App Review Notes

## Review summary

`d1v` is an AI-native workspace for project creation, repository import, project chat, file inspection, deployment visibility, usage tracking, billing surfaces, and account management.

## Authentication for review

The app supports:

- Sign in with Apple
- Sign in with Google
- Sign in with GitHub
- Sign in with Microsoft
- Email verification code
- Email and password

For App Review, authentication can be completed directly from the login screen with Sign in with Apple.

Third-party sign-in methods use the provider-hosted authentication page and return to the app after successful authorization.

No paid account is required to reach the login screen.

## Reviewer guidance

- Open the dashboard after sign-in to view projects and activity.
- Open a project to inspect overview, files, and deployment context.
- Open project chat to continue an AI-assisted project session.
- Open profile/account surfaces to review credits, settings, and account controls.
