# d1v macOS App Store 文案

## App 名称

`d1v`

## 副标题

`AI 工作台与项目控制台`

## 宣传文本

`在 Mac 上统一管理 AI 项目、代码上下文、部署状态、账单与团队工作流。`

## 关键词

`AI,开发,项目管理,代码,部署,团队协作,工作台,生产力`

## App 描述

`d1v` 是一个面向构建者、运营者与现代产品团队的 AI 原生工作台。

你可以在一个应用里完成以下核心工作：

- 创建 AI 项目，继续已有项目会话
- 导入 GitHub 仓库或接入现有代码工作流
- 在项目上下文里与 AI 持续协作，而不是停留在孤立聊天框
- 查看项目文件、代码结果、部署状态与运行信号
- 管理个人资料、额度、用量、账单、文档与社区内容

`d1v` 不是一个只会对话的通用聊天壳，而是围绕真实项目执行流程设计的工作界面。它更适合需要在桌面端持续掌控 AI 项目、远程工作空间与交付状态的个人开发者、小团队与 AI Native 产品团队。
