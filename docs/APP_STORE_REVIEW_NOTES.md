App Store Review Notes

This app supports multiple sign-in methods on iPhone:

- Sign in with Apple
- Sign in with Google
- Sign in with GitHub
- Sign in with Microsoft
- Email verification code
- Email and password

For App Review, you can complete authentication directly with Sign in with Apple from the login screen.

Google, GitHub, and Microsoft sign-in use the provider's hosted web authentication page and then return to the app after successful authorization.

If you prefer not to use a third-party account during review, Sign in with Apple is fully supported and available on the same login screen.

No paid account is required to reach the login screen.
